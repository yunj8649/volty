## Character Lock

- [ ] 1안 뽀짝 Volty와 같은 캐릭터로 보인다.
- [ ] 노란 번개형 후드가 유지됐다.
- [ ] 검은 후드티가 유지됐다.
- [ ] 둥근 흰 얼굴이 유지됐다.
- [ ] 검은 타원 눈이 유지됐다.
- [ ] 작은 입이 유지됐다.
- [ ] 분홍 볼이 유지됐다.
- [ ] 둥근 손과 발이 유지됐다.

## Style Rules

- [ ] Flat Design이다.
- [ ] SVG로 옮기기 쉬운 단순한 형태다.
- [ ] 실사가 아니다.
- [ ] 3D가 아니다.
- [ ] 그라데이션이 없다.
- [ ] 과한 그림자나 글로우가 없다.

## Brand Tone

- [ ] 선생님처럼 보이지 않는다.
- [ ] 친구처럼 보인다.
- [ ] 사용자를 혼내거나 압박하지 않는다.
- [ ] "같이 공부한다"는 감정이 남는다.
