# Volty Character Lock

> Status: Phase 1 active
> Canonical direction: 1안 뽀짝 Volty

이 문서는 인스타툰 제작을 위한 Volty 캐릭터 고정 기준이다. 새로운 그림, SVG, 프롬프트, 템플릿은 이 기준을 먼저 통과해야 한다.

## Canonical Choice

기준형은 `brand/assets/볼티_마스코트_3개_제안.png` 안의 **1안 뽀짝 Volty**다.

선택 이유:

- 앱에 이미 들어간 Volty 마스코트와 가장 가깝다.
- 둥글고 말랑한 인상이 강해서 "같이 공부하는 친구"에 맞다.
- 얼굴, 손, 발이 커서 작은 인스타툰 컷에서도 읽힌다.
- 검은 후드티와 노란 번개 후드의 대비가 강해 캐릭터 기억이 쉽다.

2안 동글 Volty와 3안 미니 Volty는 기준형이 아니다. 향후 시즌 변형, 서브 스타일, 이모티콘 변형 후보로만 둔다.

## Core Shape

Volty는 아래 요소가 동시에 보여야 한다.

- 노란 번개형 후드
- 둥근 흰 얼굴
- 검은 후드티
- 작은 검은 타원 눈
- 단순한 작은 입
- 분홍 볼
- 둥근 손
- 둥근 발
- 두껍고 부드러운 검은 외곽선

하나라도 빠지면 Volty로 인식되기 어렵다.

## Proportion Lock

정확한 픽셀 수치보다 상대 비율을 우선한다.

| Part | Rule |
|---|---|
| 전체 실루엣 | 세로로 길쭉하지 않고 둥글다 |
| 머리/후드 | 전체 인상의 60% 이상을 차지한다 |
| 얼굴 | 후드 안에서 크게 보이고 여백이 충분하다 |
| 몸통 | 머리보다 작고 검은 후드티로 보인다 |
| 손 | 얼굴 아래 양옆에 둥근 덩어리로 붙는다 |
| 발 | 작고 둥글며 몸 아래에 붙는다 |
| 눈 | 검은 타원 또는 점 형태, 하이라이트 없음 |
| 입 | 작고 단순한 곡선 또는 작은 열린 입 |

금지:

- 사람처럼 팔다리를 길게 만들기
- 눈을 크게 반짝이는 애니메이션 눈으로 바꾸기
- 후드 실루엣을 머리카락처럼 복잡하게 만들기
- 몸통을 재킷, 교복, 로브 등 다른 옷으로 바꾸기
- 손가락을 자세히 그리기

## Color Lock

기본 컬러:

| Token | Color | Usage |
|---|---|---|
| Volty Yellow | `#FFD54F` | 번개 후드, 강조 |
| Volty Amber | `#FFB300` | 번개 심볼, 작은 포인트 |
| Volty Black | `#333333` | 외곽선, 후드티 |
| Volty White | `#FFFFFF` | 얼굴, 손, 발 |
| Cheek Pink | `#FFC7B0` | 볼 |
| Paper | `#F8FAFC` | 밝은 배경 |

변형 규칙:

- 후드의 노란색은 바꾸지 않는다.
- 후드티는 기본 검정이다.
- 계절 변형은 Brand Bible에 등록한 뒤 사용한다.
- 그림자, 글로우, 그라데이션은 인스타툰 SVG에서 사용하지 않는다.

## Direction Lock

우선 필요한 방향:

- 정면
- 좌측 3/4
- 우측 3/4
- 옆모습
- 뒷모습

뒷모습에서도 노란 번개 후드와 검은 후드티가 보여야 한다.

주의:

방향 SVG는 아직 승인된 기준 자산이 없다. 새 방향 SVG는 손으로 임의 재구성하지 않고, 기준 시안 또는 승인된 원본을 트레이싱해서 만든다.

## Instagram Usage

정보 컷:

- Volty는 작게 배치한다.
- 공식, 단위, 판단 기준을 가리지 않는다.
- 좌하단 또는 우하단에 둔다.

감정 컷:

- Volty를 크게 써도 된다.
- 얼굴과 후드 실루엣이 먼저 보이게 한다.

마지막 컷:

- Volty가 사용자를 몰아붙이면 안 된다.
- "오늘도 한 문제만"처럼 작은 행동으로 끝낸다.

## Approval Checklist

새 Volty 자산은 아래를 모두 통과해야 한다.

- 1안 뽀짝 Volty와 같은 캐릭터로 보이는가?
- 노란 번개형 후드가 유지되는가?
- 검은 후드티가 유지되는가?
- 눈, 입, 얼굴 비율이 유지되는가?
- 손과 발이 둥글고 단순한가?
- 실사, 3D, 그라데이션, 과한 그림자가 없는가?
- 선생님보다 친구처럼 보이는가?
- 작은 크기에서도 Volty로 읽히는가?

하나라도 실패하면 수정 후 다시 검수한다.

## SVG Production Rule

SVG 제작은 다음 순서로만 진행한다.

1. 기준 시안 crop을 먼저 둔다.
2. 시안을 기준으로 선, 면, 비율을 트레이싱한다.
3. `templates/character-asset-review.md`로 검수한다.
4. 승인된 SVG만 `brand/svg/`의 기준 자산으로 등록한다.

금지:

- 시안을 보고 대충 비슷하게 새로 그리기
- 비율을 단순화한다는 이유로 얼굴, 후드, 몸통 형태를 바꾸기
- 기존 시안과 다른 귀여움 방향으로 재해석하기
