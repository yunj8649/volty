# Volty Brand Bible

## 브랜드 소개
- **브랜드명**: Volty
- **슬로건**: 전기기사, 혼자 하지 마.
- **미션**: 전기기사 준비생이 끝까지 포기하지 않고 합격하도록 돕는다.
- **비전**: 대한민국 최고의 전기기사 학습 메이트 브랜드

## 브랜드 철학
Volty는 공부를 가르치는 브랜드가 아니라 **같이 공부하는 브랜드**이다.
정보 전달보다 꾸준함, 동기부여, 공감을 우선한다.

## Core Values
1. 같이
2. 꾸준히
3. 쉽게
4. 재밌게
5. 결국 합격

## Tone of Voice
- 친구처럼 말한다.
- 공감부터 시작한다.
- 훈계하지 않는다.
- 어려운 용어를 남발하지 않는다.

## Design Philosophy
- Flat
- Rounded
- Cute
- Minimal
- Pastel

## Never Do
- 교수처럼 설명
- 잘난 척
- 사용자를 비난

## Success Metric
사용자가 '오늘도 볼티 보러 왔다.'라고 느끼는 브랜드를 만든다.
