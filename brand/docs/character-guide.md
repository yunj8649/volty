# Volty Character Guide
> Version: 1.0.0

# Purpose
This document defines every rule required to represent Volty consistently across illustrations, UI, social media, merchandise, websites, apps and AI-generated assets.

---

# 1. Character Identity

## Basic Profile

| Item | Value |
|---|---|
| Name | Volty |
| Nickname | 볼티 |
| Role | Study Buddy |
| Species | Electric Mascot |
| Gender | Undefined |
| Age | Undefined |
| Occupation | Electrician Exam Student |

## Character Summary

Volty is not a genius.
Volty is not a teacher.
Volty is not a mascot that simply explains information.

Volty exists because studying alone is difficult.

The purpose of Volty is to become the friend users naturally open every day until they pass.

# Character Promise

Volty is not the smartest character.

Volty is the character people want beside them until they pass.

Whenever a design or piece of content is created, ask one question:

'Does this feel like a friend studying with me?'

If the answer is no, it is not Volty.

# 18. Instagram Toon Character Rules

인스타툰에서 Volty는 선생님이 아니라 같은 수험생 친구처럼 등장한다.

## Role Per Panel

첫 컷:

- 사용자와 같은 상황에 놓인다.
- 놀라거나 당황하거나 방전된 모습이어도 된다.
- 잘난 척하며 답을 알고 있으면 안 된다.

둘째 컷:

- 헷갈리는 지점을 같이 발견한다.
- "나도 이거 틀렸어" 같은 공감형 반응을 한다.

셋째 컷:

- 해결 실마리를 찾는다.
- 설명은 짧게, 행동은 귀엽게.

마지막 컷:

- 사용자를 응원한다.
- 완벽한 합격 선언보다 "오늘도 한 문제" 같은 작은 행동으로 끝낸다.

## Instagram Pose Priority

초기에는 아래 포즈를 우선 SVG화한다.

1. 방전
2. 책상에 엎드림
3. 계산기 들기
4. 노트 보기
5. 손가락으로 짚기
6. 번쩍 이해
7. 같이 웃기
8. 충전 포즈
9. 작은 승리
10. 시험장 긴장

## Scale Rule

인스타툰 한 컷 안에서 Volty는 정보보다 커지면 안 된다.

예외:

- 첫 컷 공감 장면
- 마지막 컷 응원 장면
- 감정 표현이 핵심인 밈 컷

학습 정보가 들어가는 컷에서는 Volty를 좌하단 또는 우하단에 배치한다.

## Text Rule

Volty의 말풍선은 짧아야 한다.

권장:

- 12자 이하 한 문장
- 최대 2줄
- 친구 말투

금지:

- 긴 공식 설명
- 교재 문장 복붙
- "반드시 암기" 같은 명령형

# 19. Canonical Character Lock

The canonical Instagram toon direction is **1안 뽀짝 Volty** from `brand/assets/볼티_마스코트_3개_제안.png`.

The detailed lock rules live in `docs/character-lock.md`.

In short:

- Yellow lightning hood
- Black study hoodie
- Round white face
- Tiny black oval eyes
- Small simple mouth
- Pink cheeks
- Rounded hands and feet
- Thick rounded outline

2안 동글 Volty and 3안 미니 Volty are not the default character. They can be used only as future variant candidates after Brand Bible approval.
