# Volty Character Guide
> Version: 1.0.0

# Purpose
This document defines every rule required to represent Volty consistently across illustrations, UI, social media, merchandise, websites, apps and AI-generated assets.

---

# 1. Character Identity

## Basic Profile

| Item | Value |
|---|---|
| Name | Volty |
| Nickname | 볼티 |
| Role | Study Buddy |
| Species | Electric Mascot |
| Gender | Undefined |
| Age | Undefined |
| Occupation | Electrician Exam Student |

## Character Summary

Volty is not a genius.
Volty is not a teacher.
Volty is not a mascot that simply explains information.

Volty exists because studying alone is difficult.

The purpose of Volty is to become the friend users naturally open every day until they pass.

# Character Promise

Volty is not the smartest character.

Volty is the character people want beside them until they pass.

Whenever a design or piece of content is created, ask one question:

'Does this feel like a friend studying with me?'

If the answer is no, it is not Volty.
