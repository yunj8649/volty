# Volty Instagram Toon Roadmap

Volty 브랜드 개발은 인스타툰을 중심으로 시작한다. 앱, 웹사이트, 굿즈는 인스타툰에서 검증된 캐릭터 감정과 콘텐츠 포맷을 확장하는 순서로 진행한다.

## 원칙

- 먼저 캐릭터를 고정한다.
- 그다음 반복 가능한 콘텐츠 포맷을 만든다.
- 반응이 좋은 포맷만 시리즈화한다.
- 시리즈가 쌓이면 SVG, 템플릿, 프롬프트로 제작 속도를 올린다.
- 목표는 조회수가 아니라 "오늘도 볼티 보러 와야지"라는 습관이다.

## Phase 1. Character Lock

목표: Volty가 어떤 컷에서도 같은 캐릭터로 보이게 만든다.

현재 상태:

- 기준 방향: 1안 뽀짝 Volty
- 기준 문서: `docs/character-lock.md`
- 검수 템플릿: `templates/character-asset-review.md`
- 프롬프트 기준: `prompts/character-lock.md`
- 첫 SVG 초안: `svg/volty-base-front.svg`

할 일:

- 기준 캐릭터 1개 확정: 진행 중
- 정면, 좌우 3/4, 전신 비율 확정
- 표정 12개 확정: 초안 작성
- 기본 포즈 15개 확정: 초안 작성
- 금지 변형 목록 작성: 초안 작성
- `docs/character-guide.md` 업데이트: 진행 중

완료 기준:

- 누구나 같은 기준으로 Volty를 다시 그릴 수 있다.
- AI 생성 이미지가 나와도 무엇이 틀렸는지 판단할 수 있다.

## Phase 2. Instagram Toon System

목표: 매주 반복해서 올릴 수 있는 인스타툰 포맷을 만든다.

할 일:

- 4컷 기본 구조 확정
- 캐러셀 구조 확정
- 릴스 전환 구조 확정
- 말풍선 규칙 확정
- 첫 컷 Hook 규칙 확정
- 마지막 컷 응원 규칙 확정
- `templates/instagram-toon.md` 작성 및 개선

완료 기준:

- 주제만 있으면 4컷 초안을 10분 안에 만들 수 있다.

## Phase 3. Content Pillars

목표: 반복 가능한 시리즈를 만든다.

초기 시리즈:

- 나도 이거 틀렸어
- 오늘도 한 문제만
- 방전된 날의 공부
- 시험장에서 은근 헷갈리는 것
- 같이 충전 루틴

할 일:

- 각 시리즈의 목적 정의
- 첫 20개 소재 작성
- 소재별 Hook 문장 작성
- 저장/공유 유도 문장 작성
- `content/series-roadmap.md` 업데이트

완료 기준:

- 최소 30일치 업로드 소재가 준비된다.

## Phase 4. SVG Library

목표: 인스타툰 제작 시간을 줄이기 위해 Volty를 부품화한다.

할 일:

- 얼굴 기본형 SVG
- 눈/입 표정 SVG
- 후드/몸통 기본형 SVG
- 손 포즈 SVG
- 공부 소품 SVG
- 감정 아이콘 SVG
- 배경 패턴 SVG

완료 기준:

- 새 컷을 만들 때 매번 캐릭터를 새로 그리지 않는다.
- 기존 자산을 조합해서 빠르게 제작할 수 있다.

## Phase 5. Prompt Library

목표: AI를 써도 캐릭터가 흔들리지 않게 한다.

할 일:

- 기본 캐릭터 프롬프트 작성
- 표정 생성 프롬프트 작성
- 컷 배경 프롬프트 작성
- 금지어 목록 작성
- 검수 체크리스트 작성
- `prompts/instagram-toon.md` 업데이트

완료 기준:

- AI 산출물이 Brand Bible과 충돌하는지 빠르게 검수할 수 있다.

## Phase 6. Publishing Loop

목표: 감으로 올리지 않고 반복 개선한다.

운영 루프:

1. 소재 작성
2. 4컷 스크립트
3. SVG/템플릿 제작
4. 업로드
5. 반응 기록
6. 시리즈 개선

볼 지표:

- 저장
- 공유
- 댓글 속 공감 표현
- 다음 편 요청
- 앱/웹 유입
- 공부 루틴 전환

보조 지표:

- 좋아요
- 조회수
- 팔로워 수

## Phase 7. Brand Expansion

목표: 인스타툰에서 검증된 캐릭터를 다른 접점으로 확장한다.

확장 순서:

1. 인스타툰
2. 릴스
3. 앱 온보딩
4. 웹사이트
5. 이모티콘
6. 스티커
7. 굿즈
8. 커뮤니티

확장 조건:

- Volty의 말투가 흔들리지 않는다.
- 캐릭터 비율이 유지된다.
- 사용자가 "같이 공부한다"는 감정을 느낀다.
