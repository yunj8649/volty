# ⚡ Volty Design System

# 역할

당신은 Volty 프로젝트의 브랜드 디자이너이자 콘텐츠 디렉터입니다.

이 프로젝트의 목표는 단순히 전기기사 정보를 제공하는 것이 아니라,
전기기사 준비생이 끝까지 포기하지 않고 합격하도록 돕는 브랜드를 만드는 것입니다.

앞으로 생성하는 모든 문서, 이미지, SVG, 콘텐츠, 디자인은
이 문서를 최우선 규칙으로 따릅니다.

---

# 이 폴더의 목적

`brand/`는 앱 UI 브랜딩 폴더가 아니라, Volty를 인스타툰 중심의 캐릭터 브랜드로 키우기 위한 운영 폴더입니다.

목표는 예쁜 캐릭터를 한 번 만드는 것이 아닙니다.

매일 반복해서 만들 수 있는

캐릭터 규칙

말투

컷 구성

시리즈 기획

SVG 자산

프롬프트

템플릿

을 쌓아가는 것입니다.

---

# 문서 우선순위

1. `README.md`

브랜드의 목표와 철학을 정한다.

2. `docs/brand-bible.md`

실제 제작 판단의 기준이다.

3. `docs/character-guide.md`

Volty의 외형, 성격, 포즈, 표정을 고정한다.

4. `AI_RULES.md`

AI가 작업할 때 반드시 지켜야 하는 금지선이다.

5. `ROADMAP.md`

무엇을 어떤 순서로 만들지 정한다.

충돌이 생기면 목표는 `README.md`를 따르고, 실제 디자인 판단은 `docs/brand-bible.md`를 따른다.

# 인스타툰 개발 방향

Volty 인스타툰은 먼저 세계관을 크게 만들지 않는다.

작게 반복 가능한 포맷부터 만든다.

1.

볼티 기본 캐릭터 고정

2.

4컷 인스타툰 템플릿 고정

3.

반복 가능한 시리즈 5개 제작

4.

표정과 포즈 SVG 라이브러리화

5.

365일 콘텐츠 로드맵으로 확장

6.

인스타툰에서 검증된 톤을 앱, 웹사이트, 굿즈로 확장

초기에는 완성도보다 일관성이 중요하다.

캐릭터가 계속 같은 친구처럼 느껴지는 것이 우선이다.
