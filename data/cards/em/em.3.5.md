## 유전체를 가진 도체계의 정전용량 등

평행판 사이를 유전체로 채우면 정전용량이 커진다.

$$
C = \dfrac{\varepsilon S}{d} = \dfrac{\varepsilon_0 \varepsilon_s S}{d} = \varepsilon_s C_0\ \mathrm{[F]}
$$

$C_0$ 는 진공일 때의 값이다. **비유전율 배만큼 커진다.** 실제 콘덴서에 유전체를 넣는
이유가 이것이다 — 같은 크기로 더 많이 담을 수 있다.

왜 커지는지는 앞 단원의 흐름 그대로다. 전하 $Q$ 를 고정하면 $D$ 는 그대로인데
분극이 전계를 $1/\varepsilon_s$ 로 깎는다. 전계가 줄면 전압도 줄고($V = Ed$),
$C = Q/V$ 에서 분모가 작아지니 용량이 커진다.

다른 도체계도 마찬가지로 $\varepsilon_0$ 자리에 $\varepsilon$ 을 넣으면 된다.

| 도체계 | 정전용량 |
|---|---|
| 평행판 | $C = \dfrac{\varepsilon S}{d}$ |
| 구도체 (고립) | $C = 4\pi\varepsilon a$ |
| 동심구 | $C = \dfrac{4\pi\varepsilon}{\dfrac{1}{a} - \dfrac{1}{b}}$ |
| 동축 케이블 (길이 $l$) | $C = \dfrac{2\pi\varepsilon l}{\ln(b/a)}$ |

## 유전체를 여러 개 채울 때

이 단원의 핵심이자 최대 함정이다. **어느 방향으로 나뉘었는지**에 따라 직렬이냐
병렬이냐가 갈린다.

### 극판과 나란한 층 — 직렬

유전체가 극판과 **평행**하게 층층이 쌓여 있으면 **직렬**이다.

$$
\dfrac{1}{C} = \dfrac{d_1}{\varepsilon_1 S} + \dfrac{d_2}{\varepsilon_2 S}
\quad\Longrightarrow\quad
C = \dfrac{S}{\dfrac{d_1}{\varepsilon_1} + \dfrac{d_2}{\varepsilon_2}}
$$

전하가 한 층을 지나 다음 층으로 **이어서** 가야 하니 직렬이다. 각 층이 $C_1 = \dfrac{\varepsilon_1 S}{d_1}$,
$C_2 = \dfrac{\varepsilon_2 S}{d_2}$ 인 콘덴서 두 개를 직렬로 이은 것과 같다.

이때 두 층에서 **$D$ 가 같고** $E$ 가 다르다. 유전율이 작은 층에 전계가 세게 걸리고
전압도 많이 걸린다.

### 극판에 수직인 경계 — 병렬

유전체가 극판에 **수직**으로 좌우를 나누고 있으면 **병렬**이다.

$$
C = \dfrac{\varepsilon_1 S_1}{d} + \dfrac{\varepsilon_2 S_2}{d}
$$

전하가 왼쪽 길과 오른쪽 길로 **갈라져** 가니 병렬이다. 면적을 나눠 가진 콘덴서
두 개를 병렬로 이은 셈이다.

이때 두 쪽에서 **$E$ 가 같고** $D$ 가 다르다. 전압이 같으니 $E = V/d$ 도 같고,
유전율이 큰 쪽에 전하가 더 많이 모인다.

### 구분하는 법

헷갈리면 **전하가 지나는 길**을 그려보면 된다.

| 배치 | 접속 | 같은 것 | 다른 것 |
|---|---|---|---|
| 극판과 나란한 층 (길이 이어짐) | **직렬** | $D$, $Q$ | $E$, $V$ |
| 극판에 수직인 경계 (길이 갈라짐) | **병렬** | $E$, $V$ | $D$, $Q$ |

경계조건으로도 확인된다. 직렬 구조는 경계면이 전계에 **수직**이라 $D_n$ 연속
→ $D$ 가 같다. 병렬 구조는 경계면이 전계와 **평행**이라 $E_t$ 연속 → $E$ 가 같다.
앞 단원의 두 줄이 그대로 답을 준다.

### 절반만 채운 경우

시험에 자주 나오는 형태다. "평행판 콘덴서의 절반을 유전체로 채웠다"는 말만으로는
답이 안 나온다. **어느 방향으로 절반인지**를 그림에서 봐야 한다.

**두께의 절반**($d/2$ 만큼 층으로) — 직렬이다.

$$
C = \dfrac{S}{\dfrac{d/2}{\varepsilon_0} + \dfrac{d/2}{\varepsilon}} = \dfrac{2\varepsilon_0\varepsilon_s S}{d(1 + \varepsilon_s)}
$$

**면적의 절반**($S/2$ 만큼 좌우로) — 병렬이다.

$$
C = \dfrac{\varepsilon_0 (S/2)}{d} + \dfrac{\varepsilon (S/2)}{d} = \dfrac{\varepsilon_0 S(1 + \varepsilon_s)}{2d}
$$

두 결과가 완전히 다르다. $\varepsilon_s = 3$ 을 넣어보면 직렬은 $1.5C_0$, 병렬은
$2C_0$ 다. 같은 유전체를 같은 양만큼 넣었는데 값이 다르다 — 방향이 결정한다.

### 유전체를 넣을 때 에너지 변화

앞 단원에서 강조한 구분이 여기서도 그대로 적용된다.

**전원을 연결한 채**($V$ 일정) — $W = \dfrac{1}{2}CV^2$. $C$ 가 $\varepsilon_s$ 배가
되니 에너지도 $\varepsilon_s$ 배로 **커진다.** 전원이 전하를 더 밀어 넣어준다.

**전원을 떼어낸 후**($Q$ 일정) — $W = \dfrac{Q^2}{2C}$. $C$ 가 커지니 에너지는
$1/\varepsilon_s$ 로 **줄어든다.** 줄어든 에너지는 유전체를 안으로 빨아들이는
일로 쓰인다.

이 "빨려 들어간다"가 다음 단원의 유전체에 작용하는 힘이다.
