## 직류 송전방식

### 교류가 이겼는데 왜 직류가 다시 나오나

19세기 말 **전류 전쟁**에서 교류가 이겼다. 이유는 하나였다 — **변압기로 전압을 쉽게
바꿀 수 있다.** 직류는 그게 안 돼서 먼 거리 송전이 불가능했다.

그런데 **전력용 반도체**가 발달하면서 직류도 전압을 바꿀 수 있게 됐다. 그러자 직류의
장점이 다시 보이기 시작했고, 그게 **HVDC**(고압직류송전)다.

### 직류의 장점 — 전부 "리액턴스가 없다"에서 나온다

$$
f = 0 \;\Longrightarrow\; X_L = \omega L = 0, \quad X_C = \dfrac{1}{\omega C} = \infty
$$

**리액턴스가 아예 없다.** 여기서 장점이 줄줄이 나온다.

| 장점 | 왜 |
|---|---|
| **안정도 문제가 없다** | $P = \frac{V_sV_r}{X}\sin\delta$ 자체가 없다. 상차각이 없으니 탈조도 없다 |
| **페란티가 없다** | 충전전류가 없다 ($X_C = \infty$) |
| **충전전류가 없다** | 케이블 장거리 송전이 가능 |
| **표피효과가 없다** | 도체 단면을 다 쓴다 ($\delta = \infty$) |
| **유전체손이 없다** | $P_d = \omega CV^2\tan\delta$ 에서 $\omega = 0$ |
| **주파수가 다른 계통 연계 가능** | **비동기 연계** |
| **절연이 유리** | 최대값과 실효값이 같다 |
| 전선이 **2가닥**(또는 1가닥+대지) | 3상은 3가닥 |
| 코로나·전파장해가 적다 | |

### 절연 이득

교류는 실효값 $V$ 일 때 **최대값이 $\sqrt{2}V$** 다. 절연은 최대값에서 깨지므로,
절연 설계는 $\sqrt{2}V$ 를 견디게 해야 한다.

직류는 최대값과 실효값이 같다. 그래서 **같은 절연으로 $\sqrt{2}$ 배 높은 전압**을
보낼 수 있다.

$$
\dfrac{V_{DC}}{V_{AC}} = \sqrt{2} \approx 1.41
$$

앞 카드의 코로나에서 "DC 30, AC 21.1"이었던 것과 같은 이야기다.

### 직류의 단점

| 단점 | 왜 |
|---|---|
| **차단이 어렵다** | **영점이 없다** |
| **변압이 어렵다** | 변압기를 못 쓴다. 반도체 변환장치가 필요 |
| 변환장치가 **비싸다** | 정류기·인버터 |
| **고조파** 발생 | 변환 과정에서 |
| **무효전력 소비** | 변환장치가 먹는다 |
| 계통 구성이 복잡 | 다단자 연결이 어렵다 |

**차단이 어려운 게 결정적 약점이다.**

교류는 초당 120번 전류가 **0을 지난다**(영점). 차단기가 그 순간에 아크를 끄면 된다.
**직류는 영점이 없어서** 아크가 저절로 안 꺼진다. 강제로 영점을 만들어줘야 하고,
그게 어렵고 비싸다.

$$
\text{교류 차단} = \text{영점을 기다린다} \qquad / \qquad \text{직류 차단} = \text{영점을 \textbf{만들어야} 한다}
$$

### 직류를 쓰는 곳 — 세 가지

$$
\boxed{\text{① 장거리 대전력} \qquad \text{② 해저 케이블} \qquad \text{③ 비동기 연계}}
$$

**① 장거리** — 교류는 거리가 멀수록 $X$ 가 커져 $P = \frac{V_sV_r}{X}\sin\delta$ 가
줄고 안정도가 나빠진다. 직류는 그 제약이 없다. 대략 $600\ \mathrm{[km]}$ 를 넘으면
직류가 경제적이다(변환 설비비를 상쇄).

**② 해저 케이블** — 케이블은 정전용량이 커서 충전전류가 도체 용량을 다 먹는다
(pw.2.8). 교류로는 수십 km가 한계다. 직류는 충전전류가 없어서 거리 제한이 없다.
제주 연계선이 HVDC인 이유다.

**③ 비동기 연계** — **주파수나 위상이 다른 계통**을 연결할 수 있다. 일본은 동쪽
50Hz, 서쪽 60Hz인데 직류로 연결한다. 교류로는 절대 불가능하다.

### BTB (Back-To-Back)

거리가 0인 HVDC다. 송전이 목적이 아니라 **계통 분리**가 목적이다. 한쪽 계통의
사고가 반대쪽으로 안 넘어가고, 조류를 정확히 제어할 수 있다.

## 교류 송전방식

**현재의 표준**이다. 장점이 하나뿐인데 그게 결정적이었다.

| 장점 | |
|---|---|
| **변압이 쉽다** | 변압기 하나로 끝 |
| **차단이 쉽다** | 영점이 있다 |
| 회전기(전동기)를 바로 쓴다 | |
| 계통 구성이 자유롭다 | |

| 단점 | |
|---|---|
| **안정도** 문제 | $\delta$ 가 커지면 탈조 |
| 페란티·충전전류 | 정전용량 때문 |
| 표피효과 | 실효저항 증가 |
| 무효전력 관리 필요 | |
| 절연이 불리 | 최대값이 $\sqrt{2}$ 배 |

**3상 3선식**이 송전의 표준이다.

- 단상보다 **전선이 덜 든다**(같은 전력에 대해)
- **회전자계**를 만든다 → 전동기가 자기 기동
- 전력이 **맥동하지 않는다**(순시 전력이 일정)

## 전압별 송전방식 및 송전전압

### 한국의 전압 체계

| 전압 | 용도 |
|---|---|
| $765\ \mathrm{[kV]}$ | 초고압 간선 |
| $345\ \mathrm{[kV]}$ | 주 송전 |
| $154\ \mathrm{[kV]}$ | 송전·1차 배전 |
| $22.9\ \mathrm{[kV]}$ | **배전** (3상 4선식) |
| $380/220\ \mathrm{[V]}$ | 저압 |

### 승압의 효과 — 전부 제곱이다

| 항목 | 효과 |
|---|---|
| **전력손실** $P_l = \frac{P^2R}{V^2\cos^2\theta}$ | $\dfrac{1}{V^2}$ 로 **감소** |
| **송전용량** $P = \frac{V_sV_r}{X}\sin\delta$ | $V^2$ 로 **증가** |
| **전압강하율** | $\dfrac{1}{V^2}$ 로 감소 |
| 전선 굵기 (같은 손실) | $\dfrac{1}{V^2}$ 로 가늘게 |

$$
V \to 2V \;\Longrightarrow\; \text{손실} \to \dfrac{1}{4}, \quad \text{송전용량} \to 4\text{배}
$$

**전부 제곱**이다. 승압이 전력공학의 최강 카드인 이유다.

**대가**도 있다.

| 대가 | |
|---|---|
| **절연비 증가** | 애자 개수, 이격거리 |
| **코로나** | 복도체 필요 (pw.2.3) |
| 기기 비용 | 변압기·차단기 |
| 철탑 대형화 | |

### 스틸(Still)의 식

경제적인 송전전압을 어림하는 식이다.

$$
\boxed{V = 5.5\sqrt{0.6\,l + \dfrac{P}{100}}\ \mathrm{[kV]}}
$$

- $l$ — 송전거리 $\mathrm{[km]}$
- $P$ — 송전전력 $\mathrm{[kW]}$

**거리가 멀수록, 전력이 클수록 높은 전압**이 유리하다는 게 식에 들어 있다.

$$
\text{승압 이득}(\propto V^2) \;\text{과}\; \text{절연 비용}(\propto V) \;\text{의 균형점}
$$

pw.2.4의 **켈빈의 법칙**(전선 굵기)과 같은 성격이다. 두 비용이 반대로 움직일 때
최적점을 찾는 것이다.

**경제적 전압이지 절대 기준이 아니다.** 실제로는 표준 전압 체계에 맞춰 고른다.

## 전력전송방식에 따른 송전방식

### 전선 소요량 비교 — 계산 단골

같은 전력·같은 손실·같은 거리에서 전선이 얼마나 드는지 비교한다.

| 방식 | 전선 소요량 (단상 2선식 = 100%) |
|---|---|
| **단상 2선식** | **100%** |
| **단상 3선식** | **37.5%** |
| **3상 3선식** | **75%** |
| **3상 4선식** | **33.3%** |

$$
\text{3상 4선식} < \text{단상 3선식} < \text{3상 3선식} < \text{단상 2선식}
$$

**3상 4선식이 가장 경제적**이고, 그래서 배전에 쓴다($22.9\ \mathrm{[kV-Y]}$).

**단상 3선식이 3상 3선식보다 적다**는 게 의외의 지점이다. 중성선이 전압을 두 배로
쓰게 해주기 때문이다.

### 왜 3상인가

**전선 소요량만 보면 3상 4선식이 최고**지만, 그것 말고도 이유가 있다.

| 이유 | |
|---|---|
| **회전자계** | 3상 전동기가 스스로 기동한다 |
| **전력이 일정** | 순시 전력이 맥동하지 않는다 |
| 전선 절약 | 위 표 |
| 발전기 구조 | 같은 크기로 더 큰 출력 |

**순시 전력이 일정**하다는 게 물리적으로 중요하다. 단상은 초당 120번 전력이 0이
되는데, 3상은 세 상이 서로 메워서 합이 일정하다. 그래서 전동기가 매끄럽게 돈다.

$$
p_a + p_b + p_c = 3VI\cos\theta = \text{일정}
$$

### 다상 송전

이론적으로 상 수를 더 늘리면 전선이 더 절약된다. 그런데 **6상 이상은 설비 복잡도가
이득을 넘어서** 안 쓴다. 3상이 실용적 균형점이다.
