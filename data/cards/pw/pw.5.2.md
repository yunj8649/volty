## 옥내 배선용 재료와 기구

### 전선·케이블

고압은 **케이블이 원칙**이다.

| 케이블 | 특징 |
|---|---|
| **CV** (가교폴리에틸렌) | 주력. $90°\mathrm{C}$ |
| **CVV** | 제어용 |
| 고압 절연전선 | 애자사용 공사용 |

**나전선은 원칙적으로 못 쓴다.**

### 기구

| 기구 | 역할 |
|---|---|
| **차단기**(VCB) | 고장전류 차단 |
| **단로기**(DS) | 무부하 개폐, 점검용 |
| **부하개폐기**(LBS) | 부하전류 개폐 |
| **피뢰기**(LA) | 이상전압 방호 |
| **계기용변성기**(PT·CT) | 계측·보호 |
| **전력퓨즈**(PF) | 단락 보호 |

## 배선공사

### 방법이 셋뿐 — 저압과 다르다

$$
\boxed{\text{① 애자사용} \quad \text{② 케이블} \quad \text{③ 케이블트레이}}
$$

**저압은 금속관·합성수지관·가요관 등 여러 방법이 되는데, 고압은 셋뿐이다.**

| | 저압 | **고압** |
|---|---|---|
| 금속관 | **가능** | **불가** |
| 합성수지관 | **가능** | **불가** |
| 가요전선관 | 가능 | 불가 |
| **애자사용** | 가능 | **가능** (건조·전개된 장소만) |
| **케이블** | 가능 | **가능** |
| **케이블트레이** | 가능 | **가능** |

**"고압은 애자·케이블·케이블트레이 셋"** 이 그대로 출제된다.

### 애자사용 공사

**건조하고 전개된 장소**에만 가능하다. 저압은 어디나 되는 것과 다르다.

| 항목 | 기준 |
|---|---|
| 전선 | 고압 절연전선 이상 |
| 전선 상호 간격 | $\ge 0.08\ \mathrm{m}$ |
| **다른 시설물과 이격** | $\ge 0.15\ \mathrm{m}$ |
| 지지점 간 거리 | 조영재 면을 따르면 $\le 2\ \mathrm{m}$ |

### 이격거리 — 저압과 비교

$$
\text{저압: } 0.1\ \mathrm{m} \qquad / \qquad \text{고압: } \boxed{0.15\ \mathrm{m}}
$$

**고압이 더 멀다.** 전압이 높으니 당연하다.

무엇과 떨어져야 하나.

- 다른 고압 옥내배선
- **저압** 옥내배선
- 관등회로 배선
- **약전류전선**(통신선)
- **수관·가스관**

**약전류전선과 떨어뜨리는 이유**가 pw.3.3의 유도장해다. 그리고 **가스관**은 아크가
튀면 폭발하니 당연하다.

### 고압과 저압을 분리한다

$$
\text{고압 배선과 저압 배선은 \textbf{같은 관・덕트에 넣으면 안 된다}}
$$

절연이 뚫리면 고압이 저압 쪽으로 넘어가 **감전과 화재**가 난다. pw.4.1의 절연협조가
건물 안에서도 적용되는 셈이다.

## 옥내배선의 설계

### 전선 굵기

저압과 원칙이 같다.

$$
\text{① 허용전류} \quad \text{② 전압강하} \quad \text{③ 기계적 강도}
$$

**셋 중 가장 굵은 값**을 고른다. 다만 고압은 **단락전류에 의한 열적 강도**도 본다.

$$
A \ge \dfrac{I_s\sqrt{t}}{k}\ \mathrm{[mm^2]}
$$

단락전류가 차단될 때까지 전선이 안 녹아야 한다. pw.3.5의 단락전류 계산이 여기서
쓰인다.

### 접지

| 계통 | 접지 |
|---|---|
| 고압 기기 외함 | **보호접지** |
| 변압기 중성점 | **계통접지** |
| 피뢰기 | 피뢰기 접지 |

**접지를 나누는 이유**는 사고 시 전위가 서로 넘어가지 않게 하려는 것이다. 다만
너무 가까우면 어차피 대지를 통해 연결되니 **공통접지·통합접지**를 하기도 한다.

## 옥내배선의 시험과 검사

### 절연내력 시험 — 10분

$$
\text{규정 시험전압을 \textbf{10분간} 걸어 견뎌야 한다}
$$

**10분이라는 시간이 시험에 나온다.**

| 전로 | 시험전압 |
|---|---|
| **7 kV 이하** | 최대사용전압 $\times$ **1.5배** |
| 7 kV 초과 25 kV 이하 **중성점 다중접지** | $\times$ **0.92배** |
| 7 kV 초과 60 kV 이하 | $\times$ **1.25배** (최저 $10.5\ \mathrm{kV}$) |
| 60 kV 초과 중성점 비접지 | $\times$ **1.25배** |
| 60 kV 초과 중성점 접지 | $\times$ **1.1배** (최저 $75\ \mathrm{kV}$) |
| 60 kV 초과 **중성점 직접접지** | $\times$ **0.72배** |

**패턴이 보인다.** 중성점을 **잘 접지할수록 배수가 작다.**

$$
\text{비접지 } 1.25 \;\to\; \text{접지 } 1.1 \;\to\; \text{다중접지 } 0.92 \;\to\; \text{직접접지 } 0.72
$$

**왜 그런가.** pw.3.3에서 본 대로 **직접접지면 지락 시 전압상승이 작다**(1.3배 이하).
그러니 절연을 낮게 잡아도 되고, 시험전압도 낮아진다.

$$
\text{중성점접지의 목적 = 절연 레벨 경감} \;\longrightarrow\; \text{시험전압 배수가 작아진다}
$$

**pw.3.3의 표가 여기서 숫자로 나타난다.** 접지방식이 절연 설계를 결정한다는 게
이 배수표다.

### 절연저항 측정

**DC $1000\ \mathrm{V}$ 메거**로 잰다. 저압 500V 초과와 같은 시험전압이다.

### 그 밖의 시험

| 시험 | |
|---|---|
| **절연유 시험** | 절연내력, 산가, **DGA**(가스 분석) |
| **$\tan\delta$ 측정** | 유전정접. 열화 진단 |
| **부분방전 측정** | 절연물 속 기포 |
| **접지저항 측정** | 어스테스터 |
| **보호계전기 동작 시험** | |

$\tan\delta$ 와 부분방전은 pw.4.2와 전기자기학(em.8.1, em.3.3)에서 이미 나온 것이다.
