## 배전반의 종류

$$
\text{배전반} = \text{개폐기} + \text{계전기} + \text{계측기} + \text{모선} \;\text{을 한데 모은 것}
$$

### 구조로 나눈 분류

| 종류 | 특징 |
|---|---|
| **개방형** | 충전부 노출. 위험. 옛날 방식 |
| **폐쇄형**(큐비클) | 금속 상자에 넣음. **현재 표준** |
| **데드 프론트형** | 앞면에 충전부 없음 |
| 라이브 프론트형 | 앞면에 충전부 있음 |

**큐비클이 표준**이다.

| 장점 | 단점 |
|---|---|
| **안전** (충전부 격리) | **점검 시 정전** 필요 |
| 소형 | 내부 확인 어려움 |
| 공장 제작 → 품질 균일 | |
| 먼지·습기 차단 | |

### GIS — 초고압의 표준

$$
\text{GIS} = \text{Gas Insulated Switchgear} \;(\mathrm{SF_6} \text{ 가스 절연 개폐장치})
$$

차단기·단로기·모선을 전부 $\mathrm{SF_6}$ 가스가 든 금속 용기에 넣은 것이다.

| 장점 | 단점 |
|---|---|
| **부지가 작다** (공기 절연의 $1/10$) | 비싸다 |
| **안전** | **고장 시 수리 어렵다** |
| 날씨·오염과 무관 | $\mathrm{SF_6}$ 온실가스 |
| 소음 적음 | |

**부지 절감**이 결정적이다. $\mathrm{SF_6}$ 의 절연내력이 공기의 2~3배라(pw.4.4)
같은 절연을 훨씬 좁은 공간에서 얻는다. 도심 변전소는 GIS가 아니면 지을 땅이 없다.

### 용도로 나눈 분류

| 배전반 | 용도 |
|---|---|
| **수전반** | 한전 전기를 받는 곳 |
| **변압기반** | |
| **배전반**(피더반) | 부하로 나눠줌 |
| **전동기 제어반**(MCC) | 전동기 제어 |
| **분전반** | 저압 분기 |

## 배전반의 구성

$$
\text{모선} + \text{개폐장치} + \text{보호장치} + \text{계측장치} + \text{제어전원}
$$

### 계측·표시

| 계기 | 무엇 |
|---|---|
| **전압계**(V) | PT를 통해 |
| **전류계**(A) | CT를 통해 |
| **전력계**(W) | |
| **역률계** | |
| **주파수계** | |
| **전력량계**(Wh) | 요금 계산 |

**PT와 CT를 통해 잰다**는 게 중요하다. 고압을 직접 계기에 넣을 수 없으니 낮춰서
받는다.

$$
\text{PT 2차 } 110\ \mathrm{V}, \qquad \text{CT 2차 } 5\ \mathrm{A}
$$

pw.5.3에서 본 그 값이다.

### 제어전원

$$
\text{\textbf{직류}}(\text{축전지})
$$

**왜 직류인가.** pw.1.6에서 본 대로, **교류가 정전되어도 차단기는 반드시 열려야**
하기 때문이다. 교류로 조작 전원을 받으면 정전 시 차단기를 못 연다.

$$
\text{정전} \to \text{교류 없음} \to \text{그래도 차단기는 열려야} \to \text{\textbf{직류 축전지}}
$$

## 배전반의 운용

### 조작 순서 — 시험 단골

$$
\boxed{\text{차단: CB 개방} \to \text{DS 개방}}
$$
$$
\boxed{\text{투입: DS 투입} \to \text{CB 투입}}
$$

**순서가 반대**다.

### 왜 이 순서인가

**단로기(DS)는 전류를 못 끊는다.** 아크를 소호할 장치가 없다.

$$
\text{전류가 흐르는데 DS를 열면} \to \text{아크 발생} \to \text{단락 ・ 화재 ・ 인명 사고}
$$

그래서 **차단기로 먼저 전류를 끊고**, 전류가 0인 상태에서 단로기를 연다.

투입할 때는 반대다. **단로기를 먼저 닫아** 회로를 이어놓고, 마지막에 차단기로
전류를 흘린다.

$$
\text{DS는 항상 \textbf{무전류} 상태에서만 조작한다}
$$

| 기기 | 부하전류 | 고장전류 |
|---|---|---|
| **차단기**(CB) | 개폐 **가능** | 차단 **가능** |
| **부하개폐기**(LBS) | 개폐 가능 | **불가** |
| **단로기**(DS) | **불가** | **불가** |

pw.1.5, pw.7.1에서 반복되는 구분이다.

### 인터록 (쇄정장치)

**사람이 실수해도 사고가 안 나게** 기계적으로 막는다.

$$
\text{차단기가 \textbf{열려 있어야만} 단로기를 조작할 수 있다}
$$

pw.4.4에서 본 그것이다. 순서를 지키라고 말로 하는 게 아니라 **물리적으로 불가능하게**
만든다.

**다른 인터록도 있다.**

- **정역 운전** — 정회전과 역회전 MC가 동시에 못 붙게 (pw.5.4)
- **큐비클 문** — 차단기가 투입 중이면 문이 안 열리게
- **접지개폐기** — 차단기가 열려야만 접지 가능

### 안전 작업 순서

$$
\text{① 차단} \to \text{② 검전} \to \text{③ 접지} \to \text{④ 작업}
$$

| 단계 | 왜 |
|---|---|
| **차단** | CB → DS 순서 |
| **검전** | 정말 죽었나 확인 |
| **접지** | 잔류 전하 방전 + 오송전 대비 |
| 표지 부착 | 남이 넣지 못하게 |

**접지가 왜 필요한가.** 두 가지다.

**잔류 전하**가 있다. 케이블이나 콘덴서는 정전용량이 커서 끊어도 전하가 남아 있다.
pw.2.1의 충전전류가 만든 그것이다. 만지면 감전된다.

**오송전** 대비다. 누가 실수로 다시 넣어도 접지되어 있으면 즉시 지락으로 검출되어
차단된다.

### 운용의 원칙

| 원칙 | |
|---|---|
| **인터록을 임의로 해제하지 않는다** | 사고의 대부분이 여기서 |
| **활선 여부를 반드시 검전** | 표시등을 믿지 않는다 |
| 조작 전 계통 확인 | |
| 2인 1조 | |

**표시등을 믿지 말라**는 게 중요하다. 램프가 나갔을 수도 있고, 회로가 잘못 배선됐을
수도 있다. **반드시 검전기로 직접 확인**한다.
